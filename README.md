## Installation

first install dependencies using :

```bash
  npm i
```

run this command to create needed database:
```bash
npm run db
```

run this command for migration:
```bash
npm run migrate
```

then in ".env" file set needed datas. 

with using command below you can start program:
```bash
  npm start
```
